## Technologies used

 Prisma

 Express

 Jest

 Cors

## Installation

$ npm install

$ npx prisma generate (Just in the first time that you create your dot env)

## Running the app

# Development mode

$ npm run dev

# Create a build

$ npm run build

# Run tests

$ npm run test

# Routes and how to use

http://localhost:3333/v1/"yournumber"
